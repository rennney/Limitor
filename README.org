* Introduction

The package to perform 3+1 oscillation analysis in MicroBooNE. It is designed to set exclussion limits (find allowed regions) using Feldman-Cousins / Frequentist CLs / Wilks methods for neted hypothesis testing. 

Note: The package is a PyTorch adaptation of CPU-based TOsc package 

* Install

~Limitor~ installs in the "usual Python way".  Use of ~pip~ and a Python
virtual environment is recommended. Needs PyTorch + ROOT CERN

#+begin_example
python3 -m venv venv
source venv/bin/activiate
git clone https://github.com/rennney/Limitor.git
cd Limitor
pip install -e .
#+end_example

* Usage
